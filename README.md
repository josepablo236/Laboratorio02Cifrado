Para el archivo de las permutaciones cada permutación la pusimos en una linea y la separamos con el caracter "|" identificando cada permutación. El archivo debe llamarse "Permutaciones.txt". Por ejemplo:
P10|7143829650
P8|39860127
P4|0321
EP|13023201
IP|02461357
